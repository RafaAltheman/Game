Run Dog Run!

Regras:

  1. Para iniciar o jogo clique no enter;
  2. Utilize a barra de espaço para pular os obstáculos;
  3. A pontuação é computada com o passar do tempo;
  4. Ao colidir com um dos obstáculos, o jogo recomeçará com a pontuação zerada;
  5. Pule bem alto com o cachorro e bem longe do hidrante para não colidir!;
  6. Para reiniciar, aperte o enter novamente.
