Run Dog Run!

Bem vindos ao nosso jogo feito com HTML, CSS e JavaScript!


O que é o Run Dog Run!

Sempre tivemos a curiosidade de codar um jogo, por isso criamos o Run Dog Run!, um jogo interativo que se passa numa cidade em que um cão, para atingir seu objetivo, deve pular os obstáculos.

Como Jogar?

Para iniciar o jogo, o jogador deverá pressionar enter e seu principal objetivo é pular os hidrantes o máximo de tempo possível.

A jogabilidade é bem simples, basta o player pular os obstáculos utilizando a barra de espaço. Caso o cachorro colida com um obstáculo, o jogo reiniciará e o contador ficará zerado.
